# Bun

**All-in-one JavaScript runtime & toolkit**

## Overview

Bun is a fast all-in-one JavaScript runtime and toolkit. It includes a bundler, transpiler, package manager, and test runner. Bun is designed as a drop-in replacement for Node.js, with native support for TypeScript and JSX.

## Key Features

- **Fast**: Built from scratch using Zig and JavaScriptCore
- **All-in-One**: Runtime, bundler, transpiler, and package manager
- **Node.js Compatible**: Drop-in replacement for most Node.js apps
- **TypeScript Native**: Run TypeScript files directly
- **Fast Package Manager**: Installs packages faster than npm/yarn/pnpm

## Installation

This task installs Bun via the official installation script:
```bash
curl -fsSL https://bun.sh/install | bash
```

## Dependencies

- `curl` - Required to download the installation script

## Version Check

