# GitHub CLI (gh)

**GitHub's official command line tool**

## Overview

GitHub CLI brings GitHub to your terminal. It reduces context switching, helps you focus, and enables you to script and create your own workflows more easily. You can use it to manage pull requests, issues, repositories, and more without leaving the command line.

## Key Features

- **Pull Requests**: Create, view, merge, and review pull requests
- **Issues**: Create and manage issues from the terminal
- **Repositories**: Clone, fork, and create repos
- **Actions**: View and manage GitHub Actions workflows
- **Authentication**: Seamless authentication with GitHub
