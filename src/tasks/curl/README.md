# cURL

**Command line tool for transferring data with URLs**

## Overview

cURL is a command-line tool and library for transferring data with URLs. It supports a wide range of protocols including HTTP, HTTPS, FTP, FTPS, SCP, SFTP, and many more. cURL is used in countless applications, scripts, and tools worldwide.

## Key Features

- **Protocol Support**: HTTP, HTTPS, FTP, SFTP, SCP, and 20+ other protocols
- **SSL/TLS**: Full support for secure connections
- **Proxy Support**: HTTP, SOCKS4, and SOCKS5 proxies
- **Authentication**: Basic, Digest, NTLM, Negotiate, and more
- **Resumable Downloads**: Continue interrupted transfers
