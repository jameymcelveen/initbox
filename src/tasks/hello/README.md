# Hello World

**A simple test task**

## Overview

This is a simple test task that prints "Hello from initbox!" to the console. It's useful for testing the initbox installation system and verifying that tasks execute correctly.

## Purpose

- Test that the task execution system works
- Verify shell command execution
- Demonstrate the simplest possible task structure

## What It Does

Runs a simple echo command:
```bash
echo "Hello from initbox!"
```

## Usage

Use this task to verify your initbox installation is working correctly:
```bash
initbox install <formula-url> --tasks hello
```
