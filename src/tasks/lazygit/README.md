# lazygit

**Simple terminal UI for git commands**

## Overview

lazygit is a simple terminal UI for git commands written in Go. It provides a visual interface for common git operations, making it easier to stage files, commit changes, manage branches, and resolve merge conflicts without remembering complex git commands.

## Key Features

- **Visual Interface**: Easy-to-use terminal UI
- **Staging**: Stage individual lines or hunks
- **Branching**: Easy branch management
- **Stashing**: Visual stash management
- **Rebasing**: Interactive rebase support

This task installs lazygit via:
- **macOS**: Homebrew (`brew install lazygit`)
- **Linux**: Official release binary

## Dependencies

- `git` - Git must be installed first

## Version Check

```bash
lazygit --version
```

## Usage

Simply run `lazygit` in any git repository to open the UI.
