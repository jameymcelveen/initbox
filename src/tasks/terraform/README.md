# Terraform

**Infrastructure as Code tool**

## Overview

Terraform is an infrastructure as code tool that lets you define both cloud and on-prem resources in human-readable configuration files that you can version, reuse, and share. It supports a wide range of providers including AWS, Azure, GCP, and many more.

## Key Features

- **Infrastructure as Code**: Define infrastructure in HCL
- **Multi-Cloud**: Support for all major cloud providers
- **State Management**: Track resource state over time
- **Plan & Apply**: Preview changes before applying
- **Modules**: Reusable infrastructure components
