# kubectl

**Kubernetes command-line tool**

## Overview

kubectl is the Kubernetes command-line tool that allows you to run commands against Kubernetes clusters. You can use kubectl to deploy applications, inspect and manage cluster resources, and view logs. It's the primary tool for interacting with any Kubernetes cluster.

## Key Features

- **Cluster Management**: Deploy and manage applications
- **Resource Inspection**: View cluster state and resources
- **Debugging**: Access logs and execute commands in pods
- **Configuration**: Manage cluster configurations
- **Multiple Clusters**: Easy switching between clusters
