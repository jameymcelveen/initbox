# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-01-02

### Added

- Initial release
- CLI tool with interactive wizard
- Formula-based installation from YAML files
- Support for remote formulas (GitHub, GitLab, S3, etc.)
- 22 bundled tasks (git, node-lts, docker, python, etc.)
- Version pattern matching (semver, wildcards, ranges)
- Dry run mode for safe previews
- Cross-platform support (macOS, Linux)
- Multiple package manager support (brew, apt, npm, pip, shell, curl, git)
- Category-based task selection
- Automatic dependency resolution
- Sudo requirement support for system-level installations
